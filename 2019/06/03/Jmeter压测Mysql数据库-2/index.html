图形结果参数说明

底部参数的含义如下：

  • 样本数目：总的请求数
  • 最新样本：最后一次请求的响应时间
  • 平均：所有请求的平均响应时间
  • 偏离：响应时间变化、离散程度测量值的大小，也就是数据的分布偏差，值越小越好
  • 吞吐量：服务器每分钟处理的请求数，相当于TPS，注意单位是分钟
  • 中值：响应时间的50%百分位的值，也就是有50%的请求响应时间小于该值
