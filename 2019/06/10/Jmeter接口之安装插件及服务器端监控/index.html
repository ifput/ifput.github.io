Jmeter接口之安装插件及服务器端监控

下载插件管理器

  1. 下载Jmeter插件管理器，下载地址：
    Jmeter-plugins官方下载

    另:Jmeter所有官方插件集合

  2. 下载的plugins-manager.jar放到Jmeter文件夹的lib/ext文件夹中

  3. 重新启动Jmeter，在操作栏运行中有Plugins Manager即设置正确

下载服务器资源监控插件

  1. Jmeter Plugins Manager 界面介绍

  2. 在Available Plugins中搜索PerfMon，选中后点击Apply Changes and Restart Jmeter下载PerfMon

  3. 下载完成后在Installed Plugins中可查看下载的插件，如下图

配置服务器

  1. 服务器端也需要下载启动程序PerfMon Server Agent，下载地址：ServerAgent-2.2.3

  2. 下载完成后，上传至服务器，并进行解压

  3. 启动监听

    注意： 若是Windows则运行.bat文件，Linux则运行.sh文件

在本地配置Jmeter

  1. 添加监听器，选择jp@gc-PerfMon Metrics Collector

    注：若连接超时或者连接不上，可重新设置端口号,但需每次启动都需要运行一次才能生效

    java -jar ./CMDRunner.jar --tool PerfMonAgent --udp-port 7777 --tcp-port 7777

  2. 运行实例：

# Jmeter
