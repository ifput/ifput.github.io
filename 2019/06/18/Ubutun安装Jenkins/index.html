Ubutun安装卸载Jenkins

发表于 2019-06-18 | 分类于 3.测试理论 |

Ubutun安装Jenkins

  1. Debian软件包存储库，用于自动安装和升级。要使用此存储库，请先将密钥添加到系统中：

    wget -q -O  - https://pkg.jenkins.io/debian-stable/jenkins.io.key | sudo apt-key add  -

  2. 在安装源文件添加jenkins的路径

    sudo vi /etc/apt/sources.list

    在最下方添加以下代码：

    deb https://pkg.jenkins.io/debian-stable binary /

  3. 更新apt-get库

    sudo apt-get update

  4. 安装Jenkins

    sudo apt-get install jenkins

    有可能会报以下错误：

该问题未找不到JAVA路径，用type -p java 确定是否有安装，建立软链接，再次安装

  1. 启动Jenkins

    /etc/init.d/jenkins start

  1. 访问Jenkins 

    http://Ipaddress[具体的ip地址]:8080

Ubutun卸载Jenkins

  1. 删除服务

    sudo apt-get remove jenkins

  2. 删除安装包，注意这里如果不是ubuntu那就yum

    sudo apt-get remove --auto-remove jenkins

  3. 删除配置和数据

    sudo apt-get purge jenkins

sudo apt-get purge --auto-remove jenkins

  1. 启动 service jenkins start

  2. 重启 service jenkins restart

  3. 停止 service jenkins stop

浏览器：

  1. 关闭Jenkins http://localhost:8080/exit

  2. 重启Jenkies http://localhost:8080/restart

  3. 重新加载配置信息 http://localhost:8080/reload
