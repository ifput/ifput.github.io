发表于 2019-12-27 | 分类于 3.测试理论 |

什么是死链

死链是指服务器的地址已经改变了．无法找到当前地址位置，包括协议死链和内容死链两种形式。死链出现的原因有网站服务器设置错误；某文件夹名称修改，路径错误链接变成死链等。死链主要是点击链接或者按钮调整到404等无效界面，内容死链为由于网站地址变更后，与预期的内容不一致。这些情况都会导致网站用户流失，正常是需要SEO进行优化，特别是大版本更新时，内容需进行测试

使用注意事项：

  1. 查询的链接进来带上“/”号,如测试百度，不要直接用http://www.baidu.com，最好用http://www.baidu.com/

  2. 上面的工具都支持账号密码，但实际上很难用，Xenu由于长期未更新，自带的browser不支持现在流行的前端框架，无法输入账号密码，linebot与web Link validator支持配置输入账号密码，但我试过均为生效，所以需要登录的网站其实不好用死链工具进行检测,若有找到新的工具请评论留言

  3. 上面的工具都不知支持JavaScript内的地址检测，但目前很多都是js进行拼接的链接，所以存在局限性

  4. 其他的工具有站长工具、有浏览器插件工具，实测的效果跟在线地址等差不多，但没办法导出报告

  5. 综上建议还是用Pc端的工具，因为可以导出报告，有的还支持本地文件上传。Pc检测的结果和在线地址检测的结果一样。但就目前的前端的编码方式，上述的工具都不怎么支持，需要手动在去点击按钮或链接才能全覆盖网站的链接。
