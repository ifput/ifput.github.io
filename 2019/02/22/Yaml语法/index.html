Yaml基本语法

简介

YAML（/ˈjæməl/，尾音类似camel骆驼）是一个可读性高，用来表达数据序列的格式。

YAML是”YAML Ain’t a Markup Language”（YAML不是一种标记语言）的递归缩写。在开发的这种语言时，YAML 的意思其实是：”Yet Another Markup Language”（仍是一种标记语言），但为了强调这种语言以数据做为中心，而不是以标记语言为重点，而用反向缩略语重命名。

yaml是一种比xml和json更轻的文件格式，也更简单更强大，它可以通过缩进来表示结构，在Python库 – PyYaml。可以用之作为你自动化测试框架的配置文件或者用例文件。

PyYML安装

  1. 安装

    pip install PyYAML

  2. 使用

注意”:”后面有空格

  • load()
import yaml

yaml_str = """
name: 测试
age: 10
job: Tester
"""

y = yaml.load(yaml_str)
print y

结果：

{‘name’: ‘测试’, ‘age’: 10, ‘job’: ‘Tester’}

  • dump()
    yaml_obj ={'name': '测试',
             'age': 10,
             'job': 'Tester'
             }
z = yaml.dump(yaml_obj,default_flow_style=False)
print(z)

结果：

age: 10

job: Tester

name: “测试”

load()：将yaml格式的字符串转换成Python对象

dump()：将Python对象转换成yaml格式文档

基本语法

1.基本规则

  • 大小写敏感
  • 使用缩进表示层级关系
  • 缩进时不允许使用Tab，只允许使用空格
  • 缩进的空格数目不重要，只要相同层级的元素左对齐即可

  • 表示注释，从它开始到行尾都被忽略

2.yaml转字典

#下面格式读到Python里会是个dict
name: 测试
age: 10
job: Tester

结果：

{‘name’: ‘测试’, ‘age’: 10, ‘job’: ‘Tester’}

3.yaml转列表

# 下面格式读到Python里会是个list
- 测试
- 10
- Tester

结果：

[‘测试’, 10, ‘Tester’]

4.复合结构

#下面格式读到Python里是个list里包含dict
- name: 测试
  age: 10
  job: Tester
- name: Kobe
  age: 30

结果：

[{‘name’: ‘测试’, ‘age’: 10, ‘job’: ‘Tester’}, {‘name’: ‘Kobe’, ‘age’: 30}]

5.基本类型

#这个例子输出一个字典，其中value包括所有的基本类型
str: "Hello World"
#单引号输出最后会完整输出，不会被转移
str1: 'Hello\nWorld'
#双引号输出最后会被转义
str2: "Hello\nWorld"
int: 120
float: 3.1415
boolean: true
None: null
time: 2019-02-22t11:43:30.20+08:00
date: 2019-02-22

结果：

{‘str’: ‘Hello World’, ‘str1’: ‘Hello\nWorld’, ‘str2’: ‘Hello\nWorld’, ‘int’: 120, ‘float’: 3.1415, ‘boolean’: True, ‘None’: None, ‘time’: datetime.datetime(2019, 2, 22, 3, 43, 30, 200000), ‘date’: datetime.date(2019, 2, 22)}

6.引用

# & 被引用 *引用
name: &name 测试
tester: *name

结果：

{‘name’: ‘测试’, ‘tester’: ‘测试’}

7.强制转换

# yaml是可以进行强制转换的，用 !! 实现
str: !!str 3.14
int: !!int "123"

结果：

{‘str’: ‘3.14’, ‘int’: 123}

8. 分段

#在同一个yaml文件中，可以用 --- 来分段，这样可以将多个文档写在一个文件中
---
name: James
age: 20
---
name: Lily
age: 19

# -*- coding: utf-8 -*-
import yaml

ys = yaml.load_all(file('test.yaml', 'r'))
for y in ys:
print y

结果：

{‘age’: 20, ‘name’: ‘James’}

{‘age’: 19, ‘name’: ‘Lily’}
