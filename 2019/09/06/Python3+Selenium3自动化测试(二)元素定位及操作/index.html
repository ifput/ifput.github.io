Python3+Selenium3自动化测试(二)元素定位及操作

通过开发者工具查看页面元素

打开火狐浏览器或谷歌浏览器，按F12则进入开发者工具模式，如下图：

打开被测网页的页面，通过鼠标右击，查看元素，可以看见页面元素有不同的标签名、属性值等信息，我们需要通过标签名或者属性值等信息来定位我们的被测元素。定位元素最重要的事情是要确保定位的元素在当前页面中的唯一性

页面元素的8种定位方法：

  1. 根据id进行定位:

    find_element_by_id()

  2. 根据name进行定位

    find_element_by_name()

  3. 根据class名进行定位

    find_element_by_class_name()

PS：一般情况下，网页都会有id、name、class_name,优先推荐用这三种定位方式，因为这些属性一般是不会变动的，即使网页有更改，也不会影响脚本

  1. 根据标签名进行定位

    find_element_by_tag_name()

  2. 根据文本进行定位

    find_element_by_link_text()

  3. 根据部分文本进行定位

    find_element_by_partial_link_text()

  4. 根据Xpath路径进行定位

    find_element_by_Xpath()

  5. 根据css路径进行定位

    find_element_by_css_selector()

PS：Xpath与CSS是比较万能的定位方式，当前面几种都不能用的时候，在推荐用这些方法

定位到元素后的操作

定位到元素之后，我们会对元素进行一系列操作，如点击、输入数据，常用的元素操作有如下几种：

  1. 清空

    clear()

  2. 输入内容

    send_keys()

  3. 点击元素、按钮、超链接

    click()

  4. 提交表单内容

    submit()
    定位到元素后，获得元素的属性

  6. 文本内容

    text

针对浏览器的操作：

  2. 窗口最大化

    maximize_window()

  3. 查看网页的title值

    title

  4. 查看网页的地址值
