Python3+Selenium3自动化测试(一)环境搭建

Selenium介绍

简介：

Selenium自动化测试工具，主要是用于web应用程序的自动化UI测试，是目前主流的自动化测试工具，主要特点为开源，跨平台，支持主流的浏览器,支持多种编程语言并且支持支持分布式测试用例执行，目前版本Selenium1-Selenium3，本文采用最新Selenium3作为公司自动化测试工具，Selenium IDE是一个记录和回放工具，这是非常容易使用，但它是非常不可靠且不利于脚本参数化，本次测试采用Selenium，Webdriver进行编写脚本，并用Python中自带unittest进行框架搭建。

什么样的项目适合自动化测试：

  1. 任务测试明确，不会频繁变动
  2. 比较频繁的回归测试
  3. 软件系统界面稳定，
  4. 软件维护周期长
  5. 项目进度压力不太大
  6. 被测软件系统开发比较规范，能够保证系统的可测试性
  7. 需要在多个平台运行相同的测试案例，组合遍历型的测试、大量的重复性测试

环境搭建

Python 环境搭建

目前Python有两个版本，Python2与Python3，由于Python3不向下兼容Python2，本文用Python3进行作为开发语言。

Python3下载地址：https://www.python.org/download/releases/3.0/

下载完成后直接安装，默认勾选Python为系统环境变量，安装完成后再命令行窗口输入python显示以下信息则表示安装成功：

Python pip工具安装

pip 是一个安装和管理 Python 包的工具，通过 pip 去安装 Python 包将变得十分简单，我们将省去了搜索–查找版本–下载–安装等繁琐的过程，在Python安装的目录下Scripts文件夹内直接点击pip.exe即可安装

安装 Selenium

打开命令提示符窗口，输入pip install selenium即可完成安装，默认安装最新版本，若需要指定版本安装则pip install selenium==[指定版本号]。

查看是否完成安装可直接输入pip list，若如图显示则安装成功

安装浏览器及其插件

Selenium支持主流的浏览器，包括Firefox、Chrome、IE、Edge等，后面两种均为微软浏览器，但并不推荐使用，Selenium3目前只支持最高版本位IE9浏览器，Edge为win10自带浏览器，测试时一般用Firefox或Chrome浏览器，本案例主要用Firefox进行测试

Firefox下载地址： www.firefox.com.cn/download/

WebDriver: https://github.com/mozilla/geckodriver/releases

Chrome 下载地址：https://www.google.com/intl/zh-CN_ALL/chrome/

WebDriver: http://chromedriver.storage.googleapis.com/index.html
