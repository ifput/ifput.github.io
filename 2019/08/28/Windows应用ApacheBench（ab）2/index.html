Windows应用ApacheBench（ab）2

ab POST请求

与GET请求类似，POST请求也是直接输入命令，只不过POST请求需要加一些实体信息到body中，请求头也需要特别说明上传的实体信息的类型等，如：

ab -n 1 -p "f:\postdata.txt" -T "application/json;charset=UTF-8" -H "mmToken:3a62bb39b1444fb5839f8f29d3355454" "http://IP或域名/hmm/memberBankInfo/addOrEdit"

请求说明

-p 包含了需要POST的数据的文件，一般要配合-T使用

如postdata.txt的文件内容为：

-T 指定content-type的类型，此示例上传的为json格式的，所以类型为application/json;charset=UTF-8

-H 指定请求头的信息，此示例未需要上传一个token，所以设置为”mmToken:3a62bb39b1444fb5839f8f29d3355454”

注意事项

  1. ab的HTTP协议只有1.0版本，而我们目前的HTTP版本都为1.1

  2. ab测试时，只要服务器有响应，在Failed Request中统计就不会增加，所以在调试时需要关注是否真的执行成功

  3. 在POST中数据文件中有中文字符，在上传时会乱码，有可能导致请求失败，目前博主还未找到解决方法，有找到方法的同学请赐教一下
